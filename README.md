# Health & Fitness Tracker MVP

An Online Health & Fitness Tracker application built with the MERN stack (MongoDB, Express, React, Node.js). This platform allows users to track their activities, set goals, manage diet and workout plans, and communicate with trainers/doctors.

## üöÄ Features

- **User Authentication**: Secure login and registration with role-based access control (User, Trainer, Admin, Doctor).
- **Dashboard**: Overview of daily activities, progress toward goals, and upcoming sessions.
- **Activity Tracking**: Log steps, running, swimming, cycling, yoga, and more with calorie estimation.
- **Workout Management**: Trainers can create and assign personalized workout plans.
- **Diet Management**: Dietitians/Trainers can create and assign diet plans with meal breakdowns.
- **Goal Setting**: Set and track daily/weekly goals for steps, workouts, weight, etc.
- **Progress Visualization**: Interactive charts for weight logs and activity trends.
- **Messaging**: Real-time communication between users and health professionals.
- **Notifications**: System alerts for goal achievements and plan assignments.

## üíª Tech Stack

- **Frontend**: React, React Router, Chart.js, Tailwind CSS, Axios.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose ODM).
- **Authentication**: JSON Web Tokens (JWT), Bcrypt.js.
- **Utilities**: Concurrently, Nodemon, Express-validator.
